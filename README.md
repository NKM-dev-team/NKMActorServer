# NKM Actor Server
## About
This is a backend for a turn-based multiplayer game played on a hexagonal board.

Includes characters with abilities and statistics.

## Tech stack
- Scala
- Akka
- Akka http
- Websockets
- Scala test
- MariaDB
- docker
- traefik
